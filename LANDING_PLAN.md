## Структура секций (10 блоков)

### 1. Navbar (sticky)
- Лого + Jay**Poly**Mind (текущий стиль, белый фон)
- Навигация: якорные ссылки (Возможности, Как работает, Цены)
- **Кнопка "Войти ->"** (существующая логика goToApp)

### 2. Hero (первый экран) -- паттерн Linear
- **Сохраняем split-layout:** светлый слева / тёмный справа с лого
- Тег: "Предиктивная аналитика поведения аудитории"
- Заголовок: "Узнайте реакцию рынка -- до того, как решение принято."
- Описание: текущее (JayPolyMind моделирует поведение...)
- KPI-строка: x10 быстрее | 1000+ агентов | Часы до отчёта
- CTA: "Начать симуляцию" + "Смотреть демо" (ghost button)

### 3. Проблема -- паттерн "pain points"
- Заголовок: "Решения принимаются вслепую"
- 3 карточки (тёмные, с иконками):
  - Фокус-группы: $5-50K, недели, 8-15 человек
  - Опросы: отвечают "как надо", а не как думают
  - Публикация вслепую: PR-кризис стоит миллионы
- Акцент: красноватый/оранжевый тон для "боли"

### 4. Решение: Как это работает -- паттерн Cursor
- Заголовок: "5 шагов от документа до прогноза"
- **Вертикальный timeline** (вместо горизонтального -- лучше на мобильных):
  1. Загрузка документа -> Граф знаний
  2. Генерация агентов (Скептик, Эксперт, Инфлюенсер...)
  3. Симуляция на Twitter + Reddit
  4. Аналитический отчёт
  5. Диалог с агентами
- Каждый шаг: номер + заголовок + описание + иллюстрация/скриншот справа

### 5. Ключевые цифры -- паттерн Vercel metrics
- Animated counters при скролле:
  - x200 быстрее фокус-групп
  - x100 дешевле традиционных исследований
  - 100+ AI-агентов за один запуск
  - 5 архетипов личностей
- Мини-таблица сравнения: JayPolyMind vs Фокус-группы vs ChatGPT

### 6. Технология -- паттерн Linear "Built for purpose"
- Заголовок: "Не просто чат-бот"
- 4 feature-карточки (2x2 grid):
  - Граф знаний (Neo4j) -- сущности и связи, а не сырой текст
  - Психопрофили -- MBTI, возраст, профессия, предубеждения
  - Dual-platform -- Twitter + Reddit одновременно
  - Эволюция мнений -- сдвиг позиции под влиянием дискуссии

### 7. Демо-кейсы -- паттерн Vercel tabs
- Заголовок: "Посмотрите в действии"
- 3 карточки-кейса:
  - Запуск продукта

## Дизайн-направление

- **Стиль:** текущая тёмная тема + светлый hero split (как сейчас)
- **Шрифты:** Unbounded / Onest / JetBrains Mono (текущие)
- **Цвета:** текущая палитра (#060814, #3B82F6, #38BDF8)
- **Анимации:** scroll-triggered fade-in/slide-up, animated counters, hover-эффекты на карточках
- **Адаптивность:** mobile-first, breakpoints 640/1024/1200px (как сейчас)
- **Тон:** профессиональный, уверенный, без воды

## Технические решения

- **Файл:** заменяем содержимое `Home.vue` (маршрут `/` не меняется)
- **Роутер:** не трогаем, маршрут `/` уже ведёт на Home
- **Приложение:** `/app`, `/login` и все остальные маршруты -- НЕ ТРОГАЕМ
- **Стили:** scoped CSS, без внешних фреймворков (как сейчас)
- **Калькулятор:** реактивный Vue-компонент (ref + computed)
- **Scroll-анимации:** IntersectionObserver API (нативный, без библиотек)
- **Якорные ссылки:** smooth scroll через CSS `scroll-behavior: smooth`
